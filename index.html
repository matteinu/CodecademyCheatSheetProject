Display & Positioning, a Cheat Sheet by Matteo

Attribute & Value Positioning Display Description
position: fixed X The element is pinned to a specific spot on the page
position: absolute X The element ignores siblings elements and is positioned relative to its closest parent element that is positioned with relative or absolute. The element is removed from the document flow. The element can be positioned anywhere with positioning attributes top, left, bottom and right.
position: relative X The element is positioned relative to its original position, with offset properties to determine the actual position - without offset, the declaration will have no effect.
display: block X Block level elements take the full width of their container (with line breaks before and after) and can have their width and height manually adjusted.
display: inline X Inline elements take as little space as possible, flow horizontally, and cannot have their width and height manually adjusted.
display: inline-block X Inline-block elements can appear next to each other and can have their width and height manually adjusted.
float: left/right X How far left or right an element should float within its parent element. The width of the container must be specified or the element will assume the full width of its containing element.
clear: left/right/both/none X How an element should behave when it bumps into another element within the same containing element. Usually used in combination with elements having the CSS float property, determining on wich sides floating elements are allowed to float.
z-index X How far back or forward an element will appear when it overlaps other elements. It uses integers (positive or negative), the highest will be at the foreground.
